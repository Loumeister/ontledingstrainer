# Local Scope Contract

- Ontleedlab is the local parsing product.
- This repo owns parsing learner flow, parsing UI, local sentence presentation, local evaluator behavior, local annotation conventions, and parsing-specific teacher tooling.
- Parsing-adjacent support flows may live here only if parsing remains the main instructional mode.
- Canonical werkwoordspelling runtime does not belong here.
- Shared didactic canon, shared taxonomy governance, shared sentence rules, and shared AI instruction layers do not belong here.
- Reusable cross-product artifacts move upward through grammar-core, not sideways through local runtime growth.
- Automatic documentation sync may update only docs/auto-sync files unless a human task explicitly broadens scope.
- Legacy wording that presents werkwoordspelling as the next automatic local module should be treated as non-canonical until README copy is aligned.

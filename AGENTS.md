# Codex AGENTS.md

Welcome to the Ontleedlab repository. This project uses Codex skills for Dutch grammar pedagogy and development.

## Project Guidance

- This project is a Dutch grammar exercise application called Ontleedlab.
- We prioritize pedagogical clarity over linguistic complexity where appropriate.
- Follow the skills located in `.codex/skills/` for specific tasks.

## Codex Skills

The following skills are available in `.codex/skills/`:

- `zinsontleding-repo-inspector`: **MANDATORY FIRST STEP** for any changes to sentences or validation rules.
- `zinsontleding-constraint-sentence-author`: Used for generating new exercise items.
- `grammar-coach`: Focuses on pedagogical quality and student feedback.
- `frontend-developer`: For UI/UX changes using Tailwind and Vite.
- `test-engineer`: For strengthening automated tests.
- `technical-writer`: For documentation.
- `accessibility-auditor`: For ensuring the app is accessible.
- `whimsy-injector`: For adding engaging, student-friendly tone.

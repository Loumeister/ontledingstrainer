# Learning Analytics

Use this skill for data-informed improvement.

## Core goals
- Translate log and usage signals into actionable product/didactic changes.
- Segment mistakes by role confusion and exercise step.
- Distinguish UX friction from conceptual misunderstanding.

## Repo-specific data sources
- `interactionLog.ts`
- `usageData.ts`
- `sessionReport.ts`
- `sessionHistory.ts`

## Output format
- Findings (what happened)
- Hypothesis (why)
- Action (what to change)
- Validation plan (how to measure impact)

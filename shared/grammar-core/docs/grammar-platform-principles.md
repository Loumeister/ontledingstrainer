# Grammar Platform Principles

## Purpose
This document defines the shared product principles for the future integrated grammar-and-verb-spelling platform.

## Core proposition
The platform should treat grammatical analysis and verb spelling as one learning chain, not as disconnected skills.

The instructional order remains:
1. grammatical analysis
2. function recognition
3. rule selection
4. spelling application
5. revision and writing transfer

## Shared learning logic
One sentence should be reusable across multiple instructional modes:
- parsing
- function recognition
- verb spelling
- contrast practice
- revision
- short writing transfer

## Shared diagnosis principle
Many verb-spelling failures are caused by prerequisite grammar failures.

The platform must therefore distinguish between:
- analysis failure
- function confusion
- rule application failure
- irregular form failure
- transfer failure

## Shared content principle
The platform should grow by building a shared sentence layer with rich metadata, not by maintaining separate sentence banks in each product.

## Shared teacher principle
Teacher insight becomes more useful when grammar analysis bottlenecks and spelling bottlenecks can be read together.

## Product-specific renderer principle
The shared core should define content, rules, and governance. Product repos should define the mode-specific UI and interaction patterns.

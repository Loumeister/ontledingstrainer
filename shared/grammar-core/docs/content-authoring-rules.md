# Content Authoring Rules

## Purpose
These rules govern how shared sentences and task objects must be authored in `grammar-core`.

## Rule 1: Didactic value over volume
Do not add sentences for superficial variety alone.

Each new item must add at least one of the following:
- a new misconception target
- a new contrastive relation
- a new context that changes the reasoning path
- a new transfer function

## Rule 2: Contrast before accumulation
Contrast-rich items have priority over bulk sentence growth.

## Rule 3: One sentence, multiple uses when possible
A shared sentence should preferably support more than one mode:
- parsing
- spelling
- revision
- transfer

## Rule 4: Metadata completeness
A sentence should not enter the shared bank unless its metadata is sufficient for the intended mode(s).

## Rule 5: Function-first compatibility
Shared content must support grammatical-function-first reasoning.

## Rule 6: Natural Dutch
Sentences must be natural, age-appropriate, and underbouw-appropriate.

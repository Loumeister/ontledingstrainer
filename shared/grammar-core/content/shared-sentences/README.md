# Shared Sentences

This folder contains the first shared sentence seeds for the future integrated grammar platform.

## Purpose
These files demonstrate how one sentence can support:
- parsing
- verb spelling
- contrast work
- revision or transfer

## Authoring rule
Start small and strong.

Prefer:
- contrast-rich sentences
- natural Dutch
- clear grammatical targets
- compatibility with more than one product

Avoid:
- bulk sentence accumulation without didactic value
- purely cosmetic variation
- product-specific hidden assumptions
